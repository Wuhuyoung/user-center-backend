# user-center-backend

## 项目简介

本项目是一个简单的用户管理系统的后端，实现了用户注册、登录、查询等基础功能，如图：

## 项目用到的技术

- Java 1.8
- Maven
- MySQL



## 如何使用本项目

1. 下载源码，导入Idea，与使用Spring Boot开发项目基本相同
2. 下载前端项目：https://github.com/Wuhuyoung/user-center-frontend
